# bcadr_map_vue（GIS、openlayers、vue)

# 简介

# 一、开始

### 项目注册依赖

```
//cd到项目根目录
npm install
```

### 项目启动服务
```
npm run serve
```

### 项目打包
```
npm run build
```

### EsLint检测
```
npm run lint
```
